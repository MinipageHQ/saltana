# Aperçu du Tableau de bord

Le Tableau de bord de Stelace est une interface utilisateur personnelle et riche en fonctionnalités à partir de laquelle vous pouvez gérer et configurer votre compte Stelace.

C’est via le Tableau de bord que vous sont communiquées vos clés API. Il vous facilite également la vie en permettant à tous vos collaborateurs non techniques de réaliser de nombreuses tâches comme l’édition des contenus ou la gestion des utilisateurs sans avoir à passer par les APIs. Il est donc utile à la fois pour les développeurs et les membres non techniques de votre équipe .

TIP

Note : Les fonctionnalités et vues du Tableau de bord peuvent être restreintes en fonction des droits que vous a accordés votre administrateur ou selon le plan API (SaaS) auquel vous êtes abonné. Ainsi les développeurs ne verront pas les mêmes éléments que les membres de l’équipe support.

# Accueil

Lors de votre première connexion vous êtes automatiquement dirigé vers la page d’accueil de votre Tableau de bord. Cette page contient d’importantes statistiques d’activité ainsi que quelques raccourcis tels que l’accès à vos clés API (tant que vous n’en avez pas créées), un lien vers la documentation de Stelace ou encore vers une démonstration illustrant la flexibilité et la puissance de nos APIs.

# Statistiques d’activité

La page d’accueil de votre Tableau de bord vous permet de visualiser en un coup d’œil les indicateurs clés de performance et de succès de votre plateforme. Ces indicateurs sont actualisés en temps réel et vous permettent ainsi d’adapter votre stratégie pour développer l’activité de votre plateforme.

En haut à droite de cette zone présentant vos dernières statistiques se trouve un menu déroulant vous permettant de visualiser ces données sur une période déterminée. À titre d’exemple, vous pouvez ainsi consulter:

  • le nombre de recherches qui ont été effectuées sur votre plateforme au cours du dernier mois
  • le nombre d’utilisateurs inscrits depuis la semaine dernière
  • le nombre de ressources publiées au cours des dernières 24 heures…

Toutes ces données peuvent également être affichées en cumulé depuis le lancement de votre plateforme.

# Parcourir le Tableau de bord Stelace

Le Tableau de bord est divisé en plusieurs parties :

  • Le menu latéral permet d’accéder aux différentes sections du Tableau de bord, correspondant aux fonctionnalités offertes par les APIs de Stelace. Vous pouvez par exemple modifier les informations d’un utilisateur ou le supprimer depuis la vue "Utilisateurs" sans avoir besoin d’utiliser l’API Users.
  • Le menu déroulant en haut à gauche vous donne accès à votre compte personnel et au bouton de déconnexion.
  • Le bas de page qui est présent sur chacune des vues du Tableau de bord, vous permet d’accéder à la Documentation en ligne (lien dans l’angle droit) ou à l’accueil du Tableau de bord via à un simple clic sur l’icône centrale Stelace.
  • L’angle supérieur droit contient l’onglet de bascule d’environnement. Cet onglet est extrêmement important car il vous permet de passer de votre environnement Test à votre environnement Live. Assurez-vous d’être sur le bon environnement avant d’effectuer toute action via votre Tableau de bord.
